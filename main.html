Documentation of this PHP based crawler.

This page is mainly written to gather information about Latest News about Trump from CNN website.

This is a custom-made crawler specifically to search for the contents of Donald John Trump from CNN website and display the headlines of latest news.
Web Crawlers, sometimes called scrapers, automatically scan the Internet attempting to glean context and meaning of the content they find.
The web wouldn’t function without them.
Crawlers are the backbone of search engines which, combined with clever algorithms, work out the relevance of your page to a given keyword set.
This basic crawler – not one that scans the whole CNN website, though, but one that is able to extract all the latest fifty headlines from a CNN webpage about Donald Trump.
The crawler is made by using the most common programming language of the internet – PHP.
We will be needing a web server to run PHP. If you already own a website then you have one, or else we can launch this crawler by simply hosting it on Apache local server.
In this crawler we have used a helper class called Simple HTML DOM (this zip file).
1. First pull the HTML DOM file by using the line: include_once('simple_html_dom.php');
2. Give your target url, in this case: $target_url = “http://edition.cnn.com/;.
3. Load your target url source in DOM object using: $html->load_file($target_url);
4. Now begin parsing your DOM object against the keyword to look for specific information.

In this page, the latest news of Trump have been scrapped from the CNN website which will always return the latest 50 news of Donald Trump.
The set of information scrapped for each headlines are the Headlines' Title, Link to the actual news page and image of the headline.
On hovering over the link "READ FULL TEXT HERE", a window opens up with a preview of the news of the headline.
